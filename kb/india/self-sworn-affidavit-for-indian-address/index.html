Generate Self Sworn Affidavit for Indian Address

You can generate self sworn affidavit for Indian address using below tool. It is one of the document required for the Indian passport renewal. You can use this format for any country (USA/Canada/UK/Australia/Germany etc)

This tool works locally in your browser, no data is uploaded to the server.

Note: Try generating letter in pdf format before you fill any information, just click "Generate PDF Letter" button. If it is not generating letter then enable javascript in your browser or try using other browser.
