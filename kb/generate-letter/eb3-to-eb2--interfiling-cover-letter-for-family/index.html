EB3 to EB2 Interfiling - Single Cover Letter for Family

Generate EB3 to EB2 Interfiling Cover Letter with Barcodes, its FREE!

This tool works locally in your browser, no data is uploaded to the server.

Note: Try generating letter in pdf format before you fill any information. Select no barcodes and click "Generate PDF Letter" button. If it is not generating letter then enable javascript in your browser or try using other browser.

Instructions

  • Fill all fields.
  • Generate pdf file, review it and sign.
  • Also check barcodes are correct or not using any barcode scanner app.
  • Thats all!

Add Barcodes?

Select if you want to add barcodes to cover letter or not (default option, No)

This will add A Number and I-485 receipt number barcodes. There are six options to print barcodes on cover letter at the top, any option is fine. USCIS uses barcodes to get your file number. Cover letter without barcodes is also fine.

  • print medium barcodes (default): Prints "USCIS A Number" and "I-485 receipt number" names for respective barcodes.
  • print medium barcodes without names: Prints without USCIS A Number and I-485 receipt number names on top.
  • print small barcodes: These barcodes height is little smaller than medium size barcodes.
  • print small barcodes without names: Prints without USCIS A Number and I-485 receipt number names on top.
  • print large barcodes: These barcodes height is little bigger than medium size barcodes.
  • print large barcodes without names: Prints without USCIS A Number and I-485 receipt number names on top.

Note: Principal applicant's USCIS I-485 Receipt Number and A-number are required for printing barcodes.
